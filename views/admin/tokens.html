How to Use API Token

To create a short link using the API, include your token in the X-API-Token header:

curl -X POST {{.BaseURL}}/api/v1/links \
-H "Content-Type: application/json" \
-H "X-API-Token: YOUR_TOKEN_HERE" \
-d '{"original_url": "https://example.com"}'

Or with optional custom code:

curl -X POST {{.BaseURL}}/api/v1/links \
-H "Content-Type: application/json" \
-H "X-API-Token: YOUR_TOKEN_HERE" \
-d '{"original_url": "https://example.com", "code": "mycode"}'
